Video Game Project

This project centered around the development of a multiplayer video game featuring adaptive AI. Built from scratch in C#, the game was designed to support real-time interactions between players while incorporating intelligent non-player characters capable of reacting to player behavior. We implemented networking features for multiplayer support and developed a basic AI system. The project offered valuable experience in both low-level network programming and AI design.
